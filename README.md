# 📊 Web Analytics Portfolio Website

## 🔹 Project Overview

This project is a Web Analytics based Portfolio Website developed using the Wix platform. It is designed to showcase UI/UX projects and provide insights into website performance and user interaction.

The website combines modern design with analytics features to improve user experience and understand visitor behavior.
